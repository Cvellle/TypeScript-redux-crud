# Articles crud

Create, update, delete posts with TypeScript and React

## Live at: [https://articles1.web.app](https://articles1.web.app/)

## Stack

**1.** **TypeScript**

**2.** **React**

**3.** **Redux ^4.0.5** - (useDispatch, useSelectors)

**4.** **React Hooks**
